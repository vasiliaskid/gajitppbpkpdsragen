⚙️ Generator Gaji (Rekap Multi-Bulan)

Alat ini menyatukan data gaji bulan lalu dengan data bulan baru, agar tidak perlu server / database external yang lambat.

Panduan Penggunaan (Agar data lama tidak hilang):
Langkah 1: Jika sudah punya riwayat, masukkan file data-gaji.js lama Anda (biar datanya gabung).
Langkah 2: Ketik nama bulan yang baru dicopas (contoh: Mei 2026).
Langkah 3: Copas data Gaji bulan tersebut dari Excel (hanya baris datanya saja).
Langkah 4: Tekan Tombol Download dan lemparkan hasil data-gaji.js yang baru ke dalam folder website utama.

Pilih file data-gaji.js bulan sebelumnya agar histori bulan lalu tidak terhapus dan terbawa ke yang baru.

Status Data Terdahulu: Kosong (akan membuat data baru dimulai dari 1 bulan ini)
